# ECS RideSharing Research

## Dependencies
- python-igraph==0.9.6
- numpy==1.20.2
- prettytable==2.1.0
- pyllist==0.3
- pyyaml==5.4.1

## Instructions

### Running the simulation script
1. Navigate to root folder of this project.
2. Run the script with `python src/simulation.py`.
